# Name And Fork Notice

The name `Beratungsassistent` may be used for unchanged or derived versions of this project.

If a public instance is modified or distributed as a fork, it should make that clear to users and link to the original repository:

https://github.com/suebi76/Beratungsassistent

Recommended wording:

```text
Fork des Beratungsassistenten von Steffen Schwabe, basierend auf https://github.com/suebi76/Beratungsassistent
```

Do not imply that Steffen Schwabe operates, reviews or endorses a modified third-party instance unless that is explicitly true.

`Methodenguide` is not used here as a product mark for the repository.
